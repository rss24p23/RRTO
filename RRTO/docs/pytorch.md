# RRTO pyTorch

Get pytorch sources
```
git clone git@github.com:pytorch/pytorch.git
git checkout v1.13.1
git submodule update --init --recursive
```

build pytorch
```
# only necessary when building on an NFS share
EXTRA_DOCKER_BUILD_FLAGS='--storage-opt "overlay.mount_program=/usr/bin/fuse-overlayfs"'

make -f docker.Makefile
```

launch RRTO server (outside of docker container)
```
<path to RRTO>/bin/RRTO-rpc-server
```

launch docker container, torch
```
sudo docker run --gpus all --rm -it -v <patch-to-RRTO>/RRTO:/RRTO --ipc=host pytorch:latest
LD_LIBRARY_PATH=/RRTO/cpu REMOTE_GPU_ADDRESS=<RRTO server address> LD_PRELOAD=/RRTO/cpu/RRTO-client.so python3 /RRTO/tests/test_apps/pytorch_minimal.py
```
or under gdb supervision:
```
LD_LIBRARY_PATH=/RRTO/cpu gdb -x /RRTO/tests/gdb_client_cmds python3
(gdb) run /RRTO/tests/test_apps/pytorch_minimal.py 
```
